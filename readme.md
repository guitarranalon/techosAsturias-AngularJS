Techos de Asturias versión Angular JS
=====================================

Aplicación para comprobar los avances en el reto de los techos de Asturias, idéntica a la [versión original](http://guitarranalon.github.io/techosdeasturias/ "Ir a la versión original") pero 
utilizando Angular JS en lugar de jQuery (que también se añade para los plugins).

Conclusiones del desarrollo
---------------------------
A pesar de lo sencillo del ejemplo ya se pueden apreciar varias ventajas que, imagino, se convertirán en más importantes conforme aumente la complejidad de lo desarrollado.
Quizás lo más destacable sea el no tener que preocuparse de la selección de elementos DOM y la facilidad para gestionar el data binding (y eso que, como ya digo, es un ejemplo muy simple con datos en realidad estáticos, qué no supondrá en un caso del mundo real...).
